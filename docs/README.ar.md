## نظرة عامة

معظم تحليلات السلامة المرورية في المنطقة تعتمد على الأعداد الخام للحوادث. يتميّز هذا المشروع بدمج بيانات وزارة النقل والهيئة العامة للإحصاء الرسمية — تسجيلات المركبات، رخص القيادة، البنية التحتية للطرق، وحركة المرور بين المدن — مع إحصائيات الوفيات والإصابات، لحساب **مقاييس مخاطر معدّلة حسب التعرّض** عبر جميع مناطق المملكة الثلاث عشرة.

المنصة:
- تتنبأ بمسار الوفيات الوطني مقابل مستهدفات السلامة المرورية في رؤية ٢٠٣٠
- تحدد المناطق عالية الخطورة حسب الوفيات لكل مركبة (وليس الأعداد الخام)
- تكتشف الشذوذات في أنماط حركة المرور والحوادث الشهرية
- تُجمّع المناطق حسب الملف المخاطري لاستهداف السياسات
- تُقدّم جميع المخرجات عبر منظومة MLOps إنتاجية ولوحة معلومات Power BI ثنائية اللغة

## مصادر البيانات

جميع البيانات من مصادر حكومية سعودية رسمية ومتاحة للعموم:

| المصدر | التغطية | المستوى |
|---|---|---|
| إحصائيات النقل البري GASTAT ٢٠٢٢–٢٠٢٤ | ٥ سنوات | سنوي إقليمي، شهري للحركة |
| بوابة البيانات المفتوحة GASTAT | اتجاه وطني لعشر سنوات | سنوي |
| البيانات المفتوحة لوزارة النقل | مستشعرات الحركة، مواصفات الطرق | نقطي / إقليمي |

**المخرجات المنشورة تستخدم بيانات مُجمَّعة على المستوى الإقليمي فقط.** لا تتضمن أي بيانات على مستوى الحادثة الفردية أو بيانات شخصية.

## البدء السريع

```bash
git clone https://github.com/khaliddosari/saudi-road-safety-mlops.git
cd saudi-road-safety-mlops
pip install -r requirements.txt
python src/data/build_panel.py
```

## النماذج التحليلية

| المكوّن | المنهجية |
|---|---|
| التنبؤ بالوفيات الوطنية | Prophet + SARIMA |
| تقدير مخاطر المناطق | LightGBM + SHAP |
| اكتشاف الشذوذات الشهرية | STL + Isolation Forest |
| تجميع المناطق (Clustering) | K-Means |
| التحليل الديموغرافي | Chi-square + مؤشر التفاوت |

## منظومة MLOps

- **تتبع التجارب:** MLflow
- **خدمة النماذج:** FastAPI + Uvicorn
- **الحاويات:** Docker + docker-compose
- **التكامل المستمر:** GitHub Actions

## إقرار مشاركة البيانات

يستخدم هذا المشروع بيانات مُجمَّعة ومتاحة للعموم فقط. المصادر الأصلية: الهيئة العامة للإحصاء، ووزارة النقل والخدمات اللوجستية، واللجنة الوزارية للسلامة المرورية. لا يحتوي هذا المستودع على أي سجلات فردية للحوادث أو معلومات تعريف شخصية.
